# Library Management System

## Project Description
The Library Management System is a software application developed using Python and MySQL to manage library operations efficiently. It helps librarians maintain records of books, students, book issuance, and returns.

## Features
- Add new books
- View available books
- Search books
- Issue books to students
- Return books
- Manage student records
- Store data in MySQL database

## Technologies Used
- Python
- MySQL
- DBMS
